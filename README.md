This version (1.1) of the Polar Highlighter is developed for Mozilla Firefox (v147).
> Part of the SemEval 2026 Challenge - Task 9: Detecting Multilingual, Multicultural And Multievent Online Polarization.
# Installation
Due to github space limitations, the ONNX [model download](https://huggingface.co/coderage/PolarHighlighter_SemEval-2026/tree/main) is provided at huggingface. Simply replace the [placeholder file](https://github.com/adriandahl/PolarHighlighter/blob/main/models/polar-model/onnx/REPLACE_WITH_ONNX_MODEL.txt) with our provided model.

To enable the extension, navigate to `about:debugging` in firefox, select "This Firefox", and
"Load Temporary Add-on..." by selecting the `manifest.json` of this extension.
